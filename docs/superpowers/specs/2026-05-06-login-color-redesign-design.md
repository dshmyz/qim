# 登录页配色重设计 - 纯净中性风

## 设计目标

将登录页从过时的绿色渐变风格改为现代、纯净的中性风格，参考 Linear/Notion 等现代 SaaS 产品的设计语言。

## 配色方案

### 背景
- 主背景：`#fafafa`（极浅灰，替代原来的灰蓝渐变）
- 去掉装饰性圆形元素

### 登录卡片
- 背景：`#ffffff`（纯白）
- 圆角：`16px`
- 阴影：`0 1px 3px rgba(0,0,0,0.06), 0 4px 16px rgba(0,0,0,0.04)`（更轻、更现代）
- 内边距：`56px 48px`
- 宽度：`420px`

### Logo
- 去掉蓝色背景，直接显示 logo 图片
- 尺寸：`56px × 56px`

### 按钮
- 默认：`#1a1a1a`（接近纯黑）
- Hover：`#333333`（稍浅的深灰）
- 文字：白色
- 圆角：`8px`
- 去掉 `opacity: 0.8`
- 去掉渐变色

### 输入框
- 边框：`1px solid #e8e8e8`
- 背景：`#fafafa`
- 圆角：`8px`
- Hover/Focus 边框：`#d0d0d0`

### 链接文字（服务器设置）
- 颜色：`#666666`
- Hover：`#333333`

### 窗口控制按钮
- 保持透明/无背景

## 变更文件

- `/qim-client/src/views/Login.vue` — 主要样式修改
