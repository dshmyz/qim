启动白板优化方案

Electron 应用启动时显示白屏是因为需要等待页面加载，以下是几种优化方案

问题说明

当前应用在 main.js 中创建窗口后直接加载 URL，在页面渲染完成前会显示白色空白窗口。可以通过以下方式优化：

A

启动画面 (Splash Screen)

先显示一个小的启动画面窗口（带 Logo 和加载动画），主窗口加载完成后再切换

效果预览
Q
QIM 青雀
B

背景色 + 隐藏窗口

设置窗口背景色，先隐藏窗口，等页面加载完成后再显示，避免白屏闪烁

实现思路
// 创建窗口时设置背景色
mainWindow = new BrowserWindow({
  backgroundColor: '#e8ecf1',
  show: false, // 先不显示
  ...
})

// 页面加载完成后再显示
mainWindow.webContents.on('did-finish-load', () => {
  mainWindow.show()
})
C

内联启动 HTML

在 Electron 中先加载一个本地的小型 HTML 启动页面，包含 Logo 和加载动画，等主页面准备好再切换
